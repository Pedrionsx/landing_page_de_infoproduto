# landing_page_de_infoproduto
📌 Landing Page para Venda de Curso (Projeto de Portfólio)  Esta aplicação consiste em uma landing page desenvolvida com Python e Flask, simulando a divulgação de um curso online com foco na captação de leads.  🚀 Objetivo  Praticar desenvolvimento back-end com Flask e implementar um fluxo real de captura e tratamento de e-mails de usuári
